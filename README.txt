    Title: CDD Assignment

    Author: Daniel Polak

    This assignemnt showcases how a program runs in parallel using OpenMP.
    The program finds all twin prime numbers in range from 1 to n and counts all prime numbers.
    The program prints out all twin prime numbers and count of prime numbers.

Dependecies:
gcc v 7.5.0 or greater
Make
OpenMP

Install:
run "make -k" in directory with code.

Usage:
type "./twinPrimes 'n' 'number of threads'".
type "time ./twinPrimes 'n' 'number of threads'" for time the program runs.
